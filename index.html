How it works
Three steps. One endpoint.
Send a request, get back production-ready logo assets. No queues, no callbacks to poll.
01

Request

POST your brand name, vibe, shape, and color to the endpoint. We validate your API key and deduct credits.

02

Generate

AI generates a Swiss-modernist symbol with reductive geometry and grid discipline. Cloudinary handles post-processing and lockup.

03

Deliver

Get back hosted URLs for your symbol, logo lockup, and optional horizontal layout. Ready to use immediately.

API Reference
Single endpoint. Full control.
Everything you need to integrate logo generation into your agent or workflow.
POST /api/v1/generate

Request body

Field Type Description
api_key string Your API key
project_name string Full brand name (e.g. "NeonCat")
mark_text string Characters for lettermark/monogram. Required when shape is lettermark or monogram.
lettermark: 1–8 chars (rec. 1–4) · monogram: 1–4 chars (rec. 1–3)
vibe enum minimal · modernist · futurist
shape enum mascot · abstract · badge · lettermark · monogram
color enum black · blue · red · gold · pink · neon · purple · orange
callback_url string Optional webhook for async delivery
